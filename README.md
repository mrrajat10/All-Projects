## Projects Included

### 1. Calculator
- Implements basic arithmetic operations: add, subtract, multiply, divide  
- Handles division by zero using exception handling

### 2. Time Class
- Represents hours and minutes  
- Displays time in HH:MM format  
- Calculates ratio of hours to minutes with exception handling for zero minutes

### 3. BankAccount
- Models a simple bank account  
- Supports deposit, withdrawal, and account details  

### 4. PaintBrush
- Models a paint brush with size, color, brand, and type  
- Simulates painting using the `paint()` method

### 5. Basketball Game Score Tracker
- Tracks scores for two basketball teams  
- Allows updating scores and printing current score  

### 6. Average Temperature
- Calculates average temperature for a given week  

### 7. Sum of First n Natural Numbers

### 8. Dice Roll Average
